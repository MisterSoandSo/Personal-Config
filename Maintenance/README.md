## Setting Environent Variables
```
#Setting an Environment Variable
export VARIABLE_NAME=VALUE

#Unsetting an Environment Variable
unset VARIABLE_NAME

#Listing All Set Environment Variables
set

#Calling Environment Variables in Bash
$VARIABLE_NAME
```
